# LearnToWork

Kho học tập từ cơ bản đến nâng cao.

## Cấu trúc

- basics: nền tảng lập trình (JS/TS, Git, CLI)
- data-structures: cấu trúc dữ liệu
- algorithms: thuật toán cơ bản → nâng cao
- web: HTML/CSS/JS, React/Next
- backend: Node.js/API/DB
- projects: bài tập, mini-project
- notes: ghi chú, cheat-sheet

## Cách học
- Mỗi thư mục có README riêng.
- Viết code + bài tập nhỏ.
- Commit nhỏ, rõ message.

## Bắt đầu
1. Cài Node.js LTS + Git + VS Code
2. Tạo repo GitHub
3. Push thư mục này lên GitHub

## Bản quyền
Chỉ dùng cho mục đích học tập cá nhân.
